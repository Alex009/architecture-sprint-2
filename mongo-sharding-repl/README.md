# Первый запуск

```bash
./initialize.sh
```
После выполнения данной команды приложение полностью развернуто и заполнено данными.

# Повторные запуски

Для повторных запусков просто выполняем
```bash
docker-compose up -d 
```

# Проверка данных

Посмотреть сколько документов:
```bash
./check-counts.sh
```

Посмотреть состояние шардов:
```bash
./shard-status.sh
```
